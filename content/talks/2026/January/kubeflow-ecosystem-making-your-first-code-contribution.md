## Abstract

An introduction to the Kubeflow ecosystem followed by a demo of making your first contribution.

**Part 1: Ecosystem Overview**

- What problems Kubeflow ecosystem solves and how ?
- Key components: Trainer, Pipelines, KServe, Notebooks, SDK
- Architecture and how they connect

**Part 2: How to raise your first contribution - walkthrough demo**

- Finding a real issue and fixing it : 
     - [kubeflow/trainer](https://github.com/kubeflow/trainer) - From platform admins perspective
     - [ Kubeflow/sdk](https://github.com/kubeflow/sdk) - From data scientists perspective
- Steps to cover : 
     - Finding a good first issue
     - Setting up local development environment
     - Making a code fix and testing locally
     - Running tests, linting
     - Creating a PR and what happens next
- Attendees will leave understanding the Kubeflow ecosystem and with a clear path to making their first contribution .

Basic understanding of Kubernetes
